About This Project

The Seattle Crime Explorer is an interactive mapping application designed to make public safety data more accessible to Seattle residents and decision-makers.

It visualizes over 64,000 crime incidents reported to the Seattle Police Department, allowing users to examine spatial and temporal crime patterns. Through heatmaps, point-level inspection, filtering, and neighborhood analysis, this tool reveals emerging trends in public safety.

Key Features

  • Crime type filtering
  • Neighborhood-based analysis
  • Interactive date range filtering
  • Smooth heatmap → point transitions
  • Popups with incident details
  • Real-time summary statistics

Intended Audience

  • Seattle residents and community groups
  • Prospective residents
  • Urban planners and policymakers
  • Real estate professionals
  • Public safety researchers
